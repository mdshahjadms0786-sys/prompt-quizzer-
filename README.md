## Description
This project is a simple JavaScript quiz game that runs in the browser console. 
It uses prompt() to take user input and alert() to display feedback and final score.

## Features
- Console-based quiz game
- Uses arrays to store questions and answers
- for loop to iterate through questions
- Input handling using toLowerCase() and trim()
- Score tracking and final result display
- No HTML or CSS required

## How to Run
1. Open any web browser
2. Right click on the page and select Inspect
3. Open the Console tab
4. Copy and paste the code from quiz.js
5. Press Enter to start the quiz

## Technologies Used
- JavaScript (ES6)
- Browser Console
